Insert JavaScript Documentation into these fences. Warning: this code uses an old version of jsdoc throw several other APIS. And it will not support any
Parameters:
Name Type Attributes Description
markdownContent String The WHOLE read me file, or file that has the TOC Fence in it.
files Array.<String> Any array of fully qualified source files and may include ** globs
jsDocOptions <optional>
 Optional options passed to jsdoc-to-markdown render command see jsdoc-to-markdown
Properties
Name Type Attributes Description
heading-depth <optional>
 The initial heading depth, default 2. For example, with a value of 2 the top-level markdown headings look like "## The heading".
module-index-format <optional>
 default `table`, options are: `none`, `grouped`, `table`, or `dl`
global-index-format <optional>
 default `table`, options are: `none`, `grouped`, `table`, or `dl`
property-list-format <optional>
 default `table`, options are: `list`, or `table`
member-index-format <optional>
 default `grouped`, options are: `grouped`, or `list`
